## Typography
- **Display / headings:** Marcellus (Google Fonts, weight 400 only)
- **Body / UI:** PT Serif (Google Fonts, 400/700, italic available)
- Both load together from: `https://fonts.googleapis.com/css2?family=Marcellus:ital,wght@0,400&family=PT+Serif:ital,wght@0,400;0,700;1,400;1,700`
- Note: PT Serif body text is a choice — consider pairing Marcellus with a sans (Inter) for body instead, but only if Emma approves. Current serif-on-serif is distinctive.
